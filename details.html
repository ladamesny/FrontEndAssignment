<details> and <summary>

<details> is a disclosure widget included in HTML 5 that the user can obtain additional information or controls.We can use it to create an accordion-like widget without using javascript.

A <summary> tag is used within the <details> tag as the header of a section to be expanded.

Below we're nesting <details> elements to toggle the view of a shoppping list and subsections, which we define with <summary>tags.
